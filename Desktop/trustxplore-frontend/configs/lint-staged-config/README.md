# @repo/lint-staged-config

Shared Lint Staged configuration for the monorepo.

## Usage

### Base Configuration

```js
// lint-staged.config.js
import config from '@repo/lint-staged-config/base';

export default config;
```

### Root Configuration

```js
// lint-staged.config.js
import config from '@repo/lint-staged-config/root';

export default config;
```
